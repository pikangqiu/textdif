> 建立 2026-06-22。基于 `../doc/result_exp.md` + `../AGENT.md` §5 操作日志，逐根支柱审视"现有实验证据能否撑住论文 story"。

## 结论先行

- **承重墙 = C2（蒸馏均衡器）**：证据最满、最反直觉、护城河最深 → **定为 thesis**。
- **不承重 = C1 / C3 / Real-CE-SOTA**：精排序/漂移机制/放大/Real-CE 高度均有风险，**不让它们承重**，story 即稳。
- **可发表前提**：TAIR 主场强 + C2 硬 + Real-CE 写成"跨域一致性"而非"第二 SOTA"。**Real-CE 不用高，但微调后不能看起来像退化**。

## 逐支柱

### ✅ C2 蒸馏均衡器 —— 最硬，证据充分
- teacher 侧 before/after：结构增强①② teacher 涨(63.78→64.61/65.67) **蒸馏后回落** 65.59/66.23≈B(65.99) → 干净 before/after。
- 输入侧：E1 gate 自学到负、CharAcc 65.80≈B。
- 通用对齐：dino_repa 66.00、seg_token 64.86、REPAocr(全局)65.81，全≈B。
- student 侧 OCR：2a ~67.3、2d 68.28，**与上述全部拉开 >1 分**。
- **判定**：粗粒度结论"唯 student 侧细粒度 OCR 监督穿透蒸馏均衡器"**铁证充分**（四正交反例 + 结构增强 before/after）。**这是全文最稳的一根，做 thesis。**

### ⚠️ C3 双空间监督 —— 精排序踩了自定噪声线（最该堵的自伤）
- §0 自定规则：**"只有 ≥1 分差异才算可信信号"**、"差距<std 视为平手"。
- 但冠军精排序：
  - 2a+2d **68.74** − 2d **68.28** = **+0.46 < 1** → 平手
  - 2d **68.28** − 2a 峰值 **67.66** = **+0.62 < 1** → 平手
  - 且 2d / 2a+2d 均**单 ckpt(20000)，无 std 兜底**（#1 噪声带被砍）。
- 好的一面：det 更稳（2a+2d 78.32 vs 2d 77.03 = +1.29；2a 76.3 → 差距>1）。
- **判定**：**CharAcc 上"2a+2d>2d>2a"的精排序不可靠**。二选一：(a) 补 2d/2a+2d 噪声带(反悔#1)证明排序稳；(b) **重写 C3**——不讲"谁最强"，讲"**特征空间 + 输出空间监督都有效且可无冲突叠加(det 双新高)**"，精排序锚到更稳的 det/E2E。**推荐 (b)+局部(a)**。
- 注：B(65.99)→2a(~67.3, +1.3)、→2d(68.28, +2.3) 均 >1 分 → **"OCR-aware >> baseline 与所有非OCR对照"这条粗结论很稳**；只有 OCR-aware 内部的精排序在噪声里。

### ⚠️ C1 一步蒸馏=去字符漂移 —— 机制未隔离 + 放大坏了
- 证据仅 teacher-MS 63.78 → B 65.99 一对，**"多步vs一步"与"teacher vs 蒸馏学生"两变量混淆**；"字符漂移"是断言、未隔离。
- 缺一个干净对照：**同一模型 NFE=1 vs NFE=多步 的 CharAcc**（或多步采样字符漂移可视化）。
- **1.4B 放大当前是负证据**：日志显示"蒸馏"误跑成 flow-matching(用错 trainer)、det 66.95<0.5B；又查 teacher 从零训(配置疏漏)、CharAcc~29 平台。
- **判定**：C1 降级为"**效率入场券 + 均衡器载体**"，别强吹漂移机制；要保留机制则补 NFE 对照。**1.4B scaling 修好(正确 init+shortcut 蒸馏)或直接砍出正文**，否则被自家日志的 det 66.95 反打。

### ⚠️ Real-CE —— 当前不是"赢"，只撑"跨域一致性"
- 零样本：PSNR **17 < 13mm 输入 19.65**（观感像"越超越差"）；rec_acc 0.29–0.30 仅微高于输入 0.275。
- 真实配对微调（唯一能到 ~0.31 同口径的路）**结果未出**（合成退化版=null）。
- **判定**：现阶段 Real-CE **只支撑"OCR-aware 排序跨域保持"(B<2a<2d<2a+2d，很稳很有价值)**，**不支撑"Real-CE SOTA"**。正文写成**跨域泛化**；SOTA 主张押在未出的真实配对微调，且**微调必须把观感从"退化"救回可辩护**（识别≈0.31、PSNR 不离谱低于输入）。

### ⚠️ TeReDiff 宣称数 —— 最大外部雷
- 用 TAIR benchmark 却复现不出 TAIR 自家方法宣称数(None 49.39 vs 复现 33.53)。你超过的是"可复现 TeReDiff + 其余全部 published"。
- 辩护已完备(官方权重复现不出、det 反高 76.14>74.88、HQ 校准准、缺口纯识别端、已排除 prompt_style)，但审稿人可能是 TAIR 阵营 → **必须写到滴水不漏，最好邮件联系作者要论文权重**。
